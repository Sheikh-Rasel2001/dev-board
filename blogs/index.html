Question-1: What are the different ways to select an element in the DOM?

The different ways to select an element in the DOM are : .getElementsByClassName('demo'), .getElementById('demo'), .getElementsByTagName('demo'), .querySelector('demo'), .querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML : This tag is used to insert a new tag. innerContent : innerContent is used to insert only raw text inside an element. It returns the raw text content as a string. innerText : This property focuses on the rendered text content.

Question-3: What is event delegation in the DOM?

Event delegation is the process of handling child events through a parent event.

Question-4: What is event bubbling in the DOM?

Event bubbling is the propagation of an event that stops at its origin and goes up to the root element.
